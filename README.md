## 📌 Project Overview

Developed a production-ready Customer Intelligence System that integrates customer segmentation, churn prediction, and lifetime value estimation into a unified analytics dashboard.

The project demonstrates:
- End-to-end data pipeline development
- Feature engineering using RFM methodology
- Supervised machine learning (classification & regression)
- Model serialization and deployment with Streamlit
- Business KPI visualization using interactive dashboards

This solution showcases practical machine learning deployment for real-world customer analytics.

# 🚀 Customer Intelligence System

An end-to-end Data Science project that analyzes customer behavior using RFM Segmentation, predicts Customer Churn, and estimates Customer Lifetime Value (CLV).

## 📊 Features
- RFM Segmentation
- Churn Prediction
- CLV Prediction
- Interactive Dashboard

pip install -r requirements.txt
